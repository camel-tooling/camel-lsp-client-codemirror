CodeMirror Adapter for Language Server Protocol

This is a live demo of a CodeMirror editor in the browser, which is connected over a web socket to a hosted Language Server which supports HTML, CSS, and JS/Typescript. The CodeMirror LSP adapter which connects these two is language agnostic, so any language server should work, but are not yet enabled.

The available features of the JS/TypeScript language server are most complete, and in this demo you can see:

  • Highlighted syntax errors
  • Autocompletion with type inference
  • Function signature documentation with types
  • Hover tooltips on symbols
